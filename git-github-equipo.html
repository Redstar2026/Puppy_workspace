¿Qué es Git y GitHub?

Git

La máquina del tiempo
de tu código

Guarda cada versión de tus scripts. Puedes volver a cualquier momento del pasado con un solo comando.

git log --oneline
→ ver todos los cambios guardados

GitHub

La nube del equipo
donde vive el código

Repositorio central en internet. Todo el equipo accede, descarga y sube código desde GitHub.

github.com/tu-equipo
→ repositorio central del equipo

Code Puppy

El asistente de IA
que crea y guarda automáticamente

Crea scripts, los guarda en Git y los sube a GitHub — solo dile qué hacer en lenguaje natural.

"Guarda los cambios en GitHub"
→ Code Puppy hace el git push

Cómo funciona en 4 pasos simples

1

Crear cuenta

Regístrate en
github.com

 5 minutos
2

Crear repositorio

Una carpeta en la nube para los scripts del equipo

 2 minutos
3

Conectar local

Vincula tu carpeta local con el repositorio en GitHub

git clone [url]
4

Guardar cambios

add → commit → push

git add .
git commit -m "..."
git push
Pro tip: Code Puppy puede ejecutar todos estos pasos automáticamente con una sola instrucción en lenguaje natural.

Reglas de seguridad del equipo

SÍ subir a GitHub
Scripts .py — código Python
Scripts .sql — consultas y reportes
Documentación .md / README
Configuraciones sin credenciales
Estructuras de carpetas y plantillas
NO subir a GitHub
Datos con PII (nombres, SSN, CURP)
Archivos .csv con datos sensibles
Contraseñas, tokens o API Keys
Archivos pesados >50MB (.xlsx, dumps)
Datos de salud (HIPAA) o financieros raw
Walmart VPN o Eagle WiFi requerido para usar Code Puppy
Code Puppy opera dentro de la red de Walmart (Eagle). Todos tus datos permanecen dentro de Walmart — InfoSec lo avala.

Beneficios para el equipo

Historial completo de cambios

Cada modificación queda registrada con fecha, autora y descripción. Nunca más "¿quién cambió esto?"

Colaboración sin conflictos

Varias personas trabajan en paralelo sin pisarse. Git fusiona los cambios automáticamente.

Recuperar versiones anteriores

¿Rompiste algo? Con un comando regresas al estado anterior. Es como Ctrl+Z pero infinito.

Compartir código entre compañeras

Un repositorio centralizado. Cualquiera del equipo descarga la versión más reciente con git pull

Con Code Puppy, el equipo puede enfocarse en el análisis, no en los comandos de Git.
